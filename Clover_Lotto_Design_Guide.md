# Clover Lotto 통합 앱 디자인 가이드 및 Nano Banana 프롬프트

## 1. 디자인 철학 및 원칙

*   **직관적이고 사용자 친화적:** 모든 기능은 사용자가 쉽게 이해하고 조작할 수 있도록 설계되어야 합니다. 복잡한 정보는 시각적으로 명확하게 전달하고, 핵심 기능으로의 접근성을 높입니다.
*   **현대적이고 미니멀리즘:** 깔끔하고 정돈된 레이아웃, 충분한 여백, 그리고 현대적인 타이포그래피를 사용하여 세련된 느낌을 줍니다. 불필요한 요소는 제거하여 정보의 집중도를 높입니다.
*   **신뢰감 있고 희망적인 분위기:** 로또라는 특성을 고려하여 신뢰감을 주는 색상과 희망적인 분위기를 연출하는 요소를 적절히 사용합니다. (예: 녹색, 황금색 계열)
*   **일관된 사용자 경험:** 앱 전체에서 일관된 디자인 언어, 인터랙션 패턴, 컴포넌트를 사용하여 사용자가 혼란 없이 앱을 탐색하고 기능을 사용할 수 있도록 합니다.
*   **모바일 최적화:** 작은 화면에서도 가독성과 조작성을 확보하며, 한 손 조작을 고려한 UI 배치를 우선합니다.

## 2. 핵심 디자인 요소

### 2.1. 색상 팔레트 (Color Palette)

*   **Primary Color (클로버 그린):** `#4CAF50` (로또, 행운, 클로버를 상징하는 메인 색상)
*   **Secondary Color (골드 옐로우):** `#FFC107` (당첨, 황금을 상징하는 보조 색상)
*   **Accent Color (스카이 블루):** `#2196F3` (신뢰, 명확성을 상징하며 포인트 요소에 사용)
*   **Background Color:** `#F5F5F5` (깔끔하고 밝은 배경)
*   **Text Color (Dark):** `#212121` (주요 텍스트)
*   **Text Color (Light):** `#757575` (보조 텍스트)
*   **White:** `#FFFFFF`

### 2.2. 타이포그래피 (Typography)

*   **폰트 패밀리:** Noto Sans KR (가독성이 좋고 현대적인 느낌의 산세리프 폰트)
*   **헤드라인:** Bold, 24pt ~ 32pt (화면 제목, 주요 정보)
*   **서브 헤드라인:** Medium/SemiBold, 18pt ~ 20pt (섹션 제목)
*   **본문:** Regular, 14pt ~ 16pt (일반 텍스트, 설명)
*   **작은 텍스트:** Regular, 12pt (보조 정보, 주석)

### 2.3. 아이코노그래피 (Iconography)

*   **스타일:** Material Design Icons (Filled 또는 Outlined 스타일 중 일관되게 선택)
*   **일관성:** 모든 아이콘은 동일한 스트로크 두께와 시각적 스타일을 유지합니다.
*   **색상:** Primary Color, Accent Color, Text Color (Dark) 등을 활용하여 가독성을 높입니다.

### 2.4. UI 컴포넌트

*   **카드 기반 레이아웃:** 정보 블록을 명확하게 구분하고 시각적 계층을 형성하기 위해 카드 형태를 적극 활용합니다.
*   **하단 내비게이션 바 (Bottom Navigation Bar):** 주요 기능(홈, 내 로또, 명당, 커뮤니티, 마이페이지 등) 간의 쉬운 이동을 위해 사용합니다.
*   **플로팅 액션 버튼 (FAB):** 로또 번호 생성, 로또 용지 스캔 등 핵심적인 액션에 사용합니다.
*   **폼 요소:** 간결하고 명확한 입력 필드, 토글, 라디오 버튼 등을 사용합니다.
*   **버튼:** Primary, Secondary, Outline 등 다양한 스타일의 버튼을 정의하고 일관되게 사용합니다.

화면 상단에는 '여행 플랜 추천' 제목이 있고, 하단 내비게이션 바는 유지해줘. 화면 중앙에는 추천된 여행 플랜 목록을 카드 형태로 배치해줘. 각 여행 플랜 카드는 플랜 이름, 주요 로또 명당, 포함된 관광지/맛집 이미지, 예상 소요 시간 또는 테마를 포함해줘. 카드를 탭하면 상세 여행 플랜 화면으로 이동하며, 상세 화면은 지도 기반의 경로, 각 방문지(로또 명당, 관광지, 맛집)의 정보, 사진, 예상 소요 시간 등을 보여주도록 해줘. '새로운 플랜 추천받기' 또는 '필터링' 버튼을 상단 또는 플로팅 액션 버튼으로 배치해줘."
